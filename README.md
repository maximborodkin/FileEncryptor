# File encryptor
### ASP&#46;NET приложение на архитектуре MVC
###### Позволяет шифровать и расшифровывать тексты алгоритмом Виженера.
###### Имеет возможность загрузки исходного текста из файлов формата txt и docx
###### и сохранения результата в файлы этих форматов.
&NewLine;

Приложение выполнено в виде одностраничного сайта. Для шифровки или расшифровки нужно поместить текст в соответствующее поле либо загрузить файл с расширением txt или docx. Далее нужно вставить ключ и, если текст нужно расшифровать, включить флажок "Текст зашифрован". Если этот флажок не включен, текст будет зашифрован. Если текст вставлен в поле, для преобразования нужно нажать на зелёную кнопку "Сгенерировать". Если необходимо использовать текст из файла - синюю кнопку "Загрузить файл". Обе кнопки будут недоступны, пока не будут выполнены требования для полей. Исходный текст обязательно должен присутствовать в любом виде (в поле или в файле), ключ должен быть непустым и состоять только из букв кириллического алфавита. Валидация полей есть как на стороне клиента, так и на стороне сервера. Валидация на стороне клиента не должна позволить пользоваться сайтом с невалидными данными. Но её лекго обойти. Поэтому она продублирована на сервере. В случае возникновения ошибки в нижней части страницы появится окно с текстом этой ошибки. В случае успешного преобразования текста в правой части страницы под ним появятся две кнопки для экспорта в файлы формата txt и docx.
&NewLine;

Технологии:
- Сервер -  C#, ASP&#46;NET 5, Razor
- Клиент - HTML5, JS, CSS, Bootstrap 4
- OpenXml для работы с docx
